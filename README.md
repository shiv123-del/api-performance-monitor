# API Performance Monitor

## Overview

API Performance Monitor is a Flask-based web application that checks API availability and measures response time. It helps monitor API performance by displaying HTTP status codes and response times.

## Features

* API Health Monitoring
* Response Time Tracking
* HTTP Status Code Analysis
* User-Friendly Interface
* Real-Time API Testing

## Technologies Used

* Python
* Flask
* Requests
* HTML
* CSS

## Installation

1. Clone the repository
2. Install dependencies

pip install flask requests

3. Run the application

python app.py

4. Open in browser

http://127.0.0.1:5000

## Workflow

1. User enters an API URL.
2. Application sends an HTTP request.
3. Response status code is collected.
4. Response time is calculated.
5. Results are displayed on the dashboard.
